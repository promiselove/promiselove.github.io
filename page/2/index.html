Linux安装jmagick

安装步骤

1.sudo apt-get install imagemagick libmagickwand-dev libjmagick6-java
2.sudo cp /usr/lib/jni/libJMagick.so /usr/lib/
3.find / -name jmagick.jar(找到jmagick.jar文件)
4.将jmagick.jar 拷贝到tomcat的bin目录下
